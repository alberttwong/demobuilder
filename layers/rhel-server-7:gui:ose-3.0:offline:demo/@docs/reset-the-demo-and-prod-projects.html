<!-- HEADER --> ## Reset the demo and prod projects 1. Double-click the Terminal icon on the desktop. 1. Type `oc project prod` and press `ENTER`. 1. Type `oc delete all --all` and press `ENTER` to reset your project. 1. Type `oc project demo` and press `ENTER`. 1. Type `oc delete all --all` and press `ENTER` to reset your project. 1. Type `exit` and press `ENTER` to close the terminal window. ## Reset to the beginning 1. Type `vagrant destroy`. 2. Type `vagrant up`. ** Note: `vagrant destory` will only destory your instance of the application. It doesn't delete the image in your local vagrant repo. [Back](index.html) <!-- FOOTER -->
